# 贡献指南

感谢你愿意为 HelloDaily 贡献内容！

## 如何推荐项目

1. 在 [Issues](https://github.com/shali10/HelloDaily/issues) 中提交推荐
2. 包含以下信息：
   - 项目名称和 GitHub 链接
   - 一句话介绍
   - 为什么推荐它（有趣在哪？入门级？）

## 项目筛选标准

- ✅ 有趣、好玩、有创意
- ✅ 入门级，新手能快速上手
- ✅ 最近有活跃更新
- ❌ 过于企业化或复杂的项目
- ❌ 纯商业推广

## 格式要求

每期内容遵循以下格式：

```markdown
### 🐍 Python

1. **[项目名](https://github.com/xxx)** ⭐123
   2-3 句话的项目介绍，包括核心功能和亮点。
```

## 自动化说明

HelloDaily 每天 9:00 自动搜索 GitHub Trending + 热门项目，按语言分类生成 markdown 文件。人工推荐的项目会在下一期收录。
